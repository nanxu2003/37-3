常见问题

你们支持哪些操作系统？

我们支持主流的 Linux 发行版（CentOS, Ubuntu, Debian, Alpine）以及 Windows Server 全系列版本。同时也支持国产操作系统如麒麟、统信的运维配置。

如果没有服务器，你们能提供吗？

我们本身不直接售卖服务器硬件，但我们是多家云厂商（阿里云、腾讯云）的合作伙伴，可以为您代购并提供折扣，或者为您提供硬件采购清单建议。

数据安全如何保障？

我们签署严格的保密协议（NDA）。在运维过程中，采用堡垒机审计操作，所有敏感数据均加密存储，绝不触碰客户业务核心数据。
